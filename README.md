# Mini Project 1: Analisis Kelulusan dan Predikat Wisuda

Project ini dibuat untuk memenuhi tugas Mini Project 1 mata kuliah Dasar Pemrograman.

Program ini menganalisis data calon wisudawan dari file Excel "Data Wisudawan Kel 3". Tujuannya adalah untuk menghitung jumlah wisudawan per program studi, mengklasifikasikan grade nilai akademik, menentukan predikat kelulusan (seperti Cumlaude), dan memvisualisasikan hasilnya.
